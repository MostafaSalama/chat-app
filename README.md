# Chat App

A minimal real-time chat application built with Node.js and WebSockets.

## Features

- Real-time messaging via WebSockets
- Clean, responsive chat UI
- Broadcasts messages to all connected clients

## Getting Started

```bash
npm install
npm start
```

Then open [http://localhost:3000](http://localhost:3000) in your browser.

## Stack

- **Node.js** — HTTP server
- **ws** — WebSocket server
